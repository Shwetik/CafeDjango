
# Cafe Shop with Django

A simple Cafe website showcasing products served by the cafe and a functionality to book a reservation via Email.
## Tech Stack

**Front End:** HTML, CSS. Bootstrap

**Backend:** Python(3.7.7), Django(3.1.12)
